Project: Simple Flight Ticket Booking System in PHP with source code

About Project

Simple Flight Ticket Booking System is developed using PHP, CSS, Bootstrap, and JavaScript. Talking about the project, it contains an admin side from where a user can manage all the timetables of the flights. The Admin plays an important role in the management of this system. From the user or customer side, they can register and book the flight tickets.

About System

Talking about the features of the Simple Flight Ticket Booking System, the user or the customer can simply register to the system. Then they can log in to the system. After that, the user or the customer can select the type of travel plan they want. They also can book the type of ticket. They can for the ticket through the payment gateway. Talking about the manager or the admin can section, he/she can perform the CRUD operation. They can create new flight time and type of flight tickets. They can update them and also delete them.

The project is very much user-friendly and anyone can use it. Simple Flight Ticket Booking System in PHP helps in easy management of schedules and other records of the airline company. To run this project you must have installed virtual server i.e XAMPP on your PC (for Windows). How To Run??

After Starting Apache and MySQL in XAMPP, follow the following steps

1st Step: Extract file
2nd Step: Copy the main project folder
3rd Step: Paste in xampp/htdocs/

Now Connecting Database

4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”
5th Step: Then, click on the databases tab
6th Step: Create a database naming “airlines2” and then click on the import tab
7th Step: Click on browse file and select “airlines2.sql” file which is inside “DATABASE” folder
8th Step: Click on go.

After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/demo/hompage.html”
Then simply select roles for logging in.
